# Nvim setup
Dependencies

```
python3 install pynvim
```

Install packer: https://github.com/wbthomason/packer.nvim

# Tmux setup
```
ln -s .tmux.conf $HOME/.tmux.conf
```

Enter tmux session
```
<c-b>I 			# Install plugins
<c-b>:so ~/.tmux.conf 	# Apply config
```
